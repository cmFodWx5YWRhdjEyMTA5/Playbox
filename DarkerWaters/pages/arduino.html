Arduino Pro Mino

I found and bought some nice Pro Mini 5v boards to go smaller than standard and specific to function. Although it is important on the mac to install the correct mac driver and disable the included apple one. A very useful function to set the time is to use the RTCSet demo and online timestamp to set the time on the RTC chip, or if you want alarms, the updated 3232 library works. Nicely using the Enerlib library for sleeping the chip as explained here.
