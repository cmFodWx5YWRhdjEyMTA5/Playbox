Downloading files

For my PicSync application I needed to do some file management, so decided to use the download facility of the browser to rename the dropped image files and put them somewhere organised for the user to then place in their images folder. This had a few problems as I wanted to download many files and the browsers kept asking permission. Which sucks. I did to a .zip file option using jsZip which worked OK but found that in Safari downloads suck too.

  • Downloading in Safari can use Downloadify but it isn't great to be honest.
  • Filesaver allows a browser to save a file as a download. Seems to work nicely.
  • Web QR is a nice port of google's x-crossing for creating QR codes.
  • Exif JS is a nice library for reading EXIF data in javascript.
